# Verification

このディレクトリは、ハンズオン実行証跡の置き場所です。

## 現在の状態

2026-06-17時点では、スクリプト化された教材をprivate sourceに追加した段階です。AWSリソースを作成する実行検証は、別IssueでCEO承認後に実施します。

## 記録するもの

AWS実行検証を行うときは、以下を記録します。

- 実行Issue
- 実行日時
- 実行者
- リージョン
- 使用したスクリプト
- 期待結果
- 実結果
- 画面キャプチャ保存先
- クリーンアップ結果
- 残リスク

## 注意

スクリーンショットには、認証情報、アカウントID、メールアドレス、Slack Webhook URL、個人情報を含めないでください。
